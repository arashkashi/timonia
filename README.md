# How?
### Activate envirnmant
```sh
source ./bin/activate
```
### Deactivate environment
```sh
deactivate
```

### Run tool
```sh
python main.py --help
````

### Run Tests
Navigate to tests folder and run below:
```sh
python -m unittest discover -v
```

### Add Test
follow the examples of this folder
```sh
tests/test_example
```
